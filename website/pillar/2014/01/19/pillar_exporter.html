Pillar converter ∞

19 Jan 2014
website pillar
BenjaminVanRyseghem

I am happy announcing my implementation of a Pillar converter for Jekyll. This way, posts and documentation can be written using the Pillar syntax.

The converter is based on a Ruby script calling a Pharo image. The Pharo image converts the Pillar input are gives it back to Jekyll. All the file matching *.pier are thus converted.

Few things to know about this exporter:

  • First the content is exported in markdown, then in html using kramdown,
  • Since the Pillar syntax is conflicting with the Liquid syntax, you may need to use the tags raw and endraw,
  • Inside the raw blocks, the string delimited by " are splitted on space. You'd better use ',
  • The character " must be escaped with \,
  • The character \ should be escaped using its ASCII value &#92;.

Thank you Damien Cassou for your great work on Pillar !
